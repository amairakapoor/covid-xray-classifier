# COVID X-ray Classifier

## 🔍 Overview
A deep learning model that classifies chest X-ray images to detect potential COVID-19 infections.

## 💡 Problem
Early detection of COVID-19 through imaging can assist healthcare professionals in faster diagnosis.

## 🚀 Solution
This project uses CNN-based deep learning models to classify X-ray images into COVID-positive or negative categories.

## ✨ Features
- Image classification using CNN  
- High accuracy prediction model  
- Easy-to-use interface  

## ⚙️ Tech Stack
- Python  
- TensorFlow / Keras  
- OpenCV  
- NumPy
